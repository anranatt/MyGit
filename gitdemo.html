多版本的回退
1.回退到之前的指定版本 ，先使用$ git log查看之前的版本提交日志 找到想好回退的版本id,接着使用$ git reset --hard commit id
2.返回未来的版本，先使用$ git reflog查看每一次命令记录，找到 你想要回到的未来版本id,接着使用$ git reset --hard commit id
